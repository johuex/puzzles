# Decode string

# Description
Дана закодированная строка следующего формата: 'k[text]', где k -- число повторений строки

Строка гарантированно имеет корректный формат без пробелов и скобок

Необходимо декодировать строку


# Examples
```
3[a]2[bc] => aaabcbc

3[a2[c]] => accaccacc

2[abc]3[cd]ef  => abcabccdcdef
```
